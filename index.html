Términos y etiquetas básicas

Función:
En programación, una función es un bloque de código reutilizable que realiza una tarea específica. Puede recibir argumentos y devolver un resultado.
Let:
"Let" es una palabra clave en JavaScript que se utiliza para declarar variables de ámbito local. Las variables declaradas con "let" tienen un alcance limitado al bloque en el que se definen.
Const:
"Const" es una palabra clave en JavaScript que se utiliza para declarar constantes. Las constantes son variables cuyo valor no puede cambiar una vez que se les ha asignado un valor inicial.
Etiqueta h1:
La etiqueta h1 se utiliza para definir un encabezado de nivel 1 en HTML. Es el encabezado más grande y generalmente se utiliza para el título principal de una página.
Etiqueta p:
La etiqueta p se utiliza para definir un párrafo en HTML. Se utiliza para agrupar texto o contenido similar en un bloque de párrafo.
Etiqueta img:
La etiqueta img se utiliza para insertar una imagen en HTML. Se debe especificar el atributo `src` para indicar la ruta de la imagen.
